This porgram will do nothing more than the implementation of OOP to the Upper-Intermediate level Student Management System .

Features Implemented :

All Features from Beginner to Upper Intermediate level



Encapsulation ;

Student attributes (id, marks, attendance) are private.

Controlled access via getters/setters.

Inheritance ;

student class inherits from Person (base class with name and age).

Polymorphism ;

Virtual function displayInfo() in Person.

Overridden in student to show detailed student info.

Used in displayStudentInfo() for polymorphic behavior.

Friend Function ;

exportsdata() allows external access to private members for exporting student data.

Extended to work with ofstream for file export.
